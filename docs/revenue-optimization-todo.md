## 风险管控
```markdown
【简化原则】发现问题即关闭付费功能，免费服务不受影响
【技术风险】支付失败或积分错乱 → 立即暂停新付费，修复后恢复
【用户体验】投诉或退款率 >10% → 调整定价或暂停相关功能
【法律合规】任何地区合规问题 → 该地区立即停止收费
【止损机制】连续3周未达成阶段KPI → 暂停该功能开发，保持现状
```
